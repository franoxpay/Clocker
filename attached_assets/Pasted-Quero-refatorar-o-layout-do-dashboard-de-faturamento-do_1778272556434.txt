Quero refatorar o layout do dashboard de faturamento do painel admin para deixá-lo mais compacto, alinhado e profissional visualmente.

IMPORTANTE:
Não alterar regras de negócio.
Não alterar cálculos.
Não alterar endpoints.
Mudança é principalmente de UI/UX/layout/responsividade.

PROBLEMAS ATUAIS:

1. O conteúdo está muito colado no menu lateral esquerdo.

* O grid começa praticamente em cima da sidebar.
* Falta padding/margin horizontal.
* O dashboard parece “espremido”.

2. Os cards estão grandes demais.

* Muito espaço vazio.
* Altura exagerada.
* Pouca densidade de informação.

3. O layout desperdiça espaço vertical.

* Em telas grandes deveria mostrar mais conteúdo acima da dobra.
* Hoje precisa scroll cedo demais.

4. Os gráficos e tabelas ocupam espaço excessivo.

* O gráfico de “Novos Usuários” está enorme mesmo sem dados.
* O donut ocupa muito espaço vazio.

5. Visualmente o dashboard parece desalinhado.

* Espaçamentos inconsistentes.
* Cards com alturas diferentes.
* Grid pouco uniforme.

OBJETIVO VISUAL:

Quero um dashboard estilo:

* Stripe Dashboard
* Vercel Analytics
* BetterStack
* Linear
* Supabase Studio

Ou seja:

* compacto
* limpo
* denso
* moderno
* profissional
* com muito dado visível sem poluição
